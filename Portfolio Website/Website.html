Portfolio Website

WWW Authoring Project

The portfolio website is an opportunity to showcase my skills as a web developer. I have also showcased my software development on the projects page. I created the website using my knowledge and understanding of HTML, CSS, and frameworks. The specific framework I used was Unsemantic.


I used HTML to structure the pages and add the content. For example, I used HTML to create a nav bar and add elements to it, however at this stage there is no design it is simply just a list with links.


Once the structure was finished and the content added (sort of added, there was a lot of lorem) I began to implement the design of the page using CSS and Unsemantic to help with the layout. CSS is giving a HTML tag a set of rules to follow that contribute to its design. The order of rules can some times be mixed up and not work exactly how you have planned it to so to avoid this I cleared all pre existing CSS on tags using a CSS reset from meyerweb.com.
